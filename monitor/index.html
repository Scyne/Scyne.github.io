Setup Instructions

Backend Options

Option 1: Free Cloud Solutions

  • Railway: Free tier with 512MB RAM, perfect for small monitoring apps
  • Render: Free tier with automatic deploys from GitHub
  • Fly.io: Free allowance for small applications
  • Supabase: Free PostgreSQL database with real-time subscriptions

Option 2: Your Ubuntu Server (hive.scyne.com)

Since you have an Ubuntu server available, you can:

  • Deploy a simple Node.js/Express backend
  • Use SQLite for lightweight data storage
  • Set up nginx as a reverse proxy
  • Use PM2 for process management

macOS Permissions & Setup

  1. System Preferences → Security & Privacy → Privacy
  2. Enable "Accessibility" for your monitoring app
  3. Enable "Screen Recording" if needed
  4. For network monitoring, enable "Full Disk Access"

Quick Start Commands

# Install dependencies
npm install express socket.io cors

# Basic server setup
const express = require('express');
const http = require('http');
const socketIo = require('socket.io');

const app = express();
const server = http.createServer(app);
const io = socketIo(server);

server.listen(3000, () => {
  console.log('Server running on port 3000');
});
